# 時系列解析レポート: ARモデル・状態空間モデルによる音楽音声生成・ノートストレッチ

## 概要

ARモデル・状態空間モデルを音楽音声に適用し、**ノートのタイム・ストレッチ（音長再構成）** を実現するシステムを構築する。単純な予測タスクを序論として、応用として音楽制作で実用的なノートストレッチャーを実装・評価する。

---

## レポート構成

### 80点パート（基礎）: 弦楽器単音の時系列予測

- **データ**: NSynth Dataset（Google）の violin / guitar 単音サンプルを使用
- **手法**: AR(p)モデル、状態空間モデル（Kalmanフィルタ）
- **タスク**: 音声フレームの時系列予測（前半で学習 → 後半を推定）
- **評価**: log-spectral distance、セグメンタルSNR

### 40点パート（進んだ分析）: ARベースのノートストレッチャー

- **データ**: Splice等から取得した楽器単音サンプル（各ノートは局所定常）
- **タスク**: 特定のノートの音長を変更して自然に聴こえるよう再構成
- **比較対象**: 位相ボコーダ（librosa.effects.time_stretch）

---

## 実装方針

### パイプライン

```
入力音声
  → onset検出でattack / sustain / releaseに分割
  → sustainのみ抽出・正規化
  → log-melスペクトログラムに変換
  → フレームごとにAR/状態空間モデルをフィット
  → 自己回帰的にフレームを延長（or 短縮）
  → Griffin-Limで波形に戻す
  → attack + 延長sustain + release を再結合
  → 出力
```

### 振幅包絡線の分離

音声信号は振幅包絡線 e(t) が時間変化するため、弱定常性の仮定が崩れる。対処として加法分解を使う：

```
log x(t) = log e(t) + log s(t)
```

- **e(t)**: フレームごとのRMSまたはHilbert変換で推定（パワーエンベロープ）
- **s(t)**: 正規化後の残差信号 → これにARモデルを適用
- **生成時**: 生成した log s(t) に予測した log e(t) を加算して再結合
- sustainはe(t)がほぼフラットなので正規化だけで対処可能
- attackとreleaseは非定常なので触らない（分離設計で回避）

### スペクトログラムドメインのモデリング

生波形ではなくlog-melスペクトログラムのフレーム列を時系列とみなしてモデリングする：

- **ビンごと独立AR**: 各メルビンを別々の時系列としてAR(p)
- **VAR（ベクトル自己回帰）**: ビン間の相関も係数行列で捉える（パラメータ爆発に注意、K=80ならK²×p個）
- **状態空間（動的因子モデル）**: 高次元観測を少数の潜在状態から生成する仮定でパラメータ削減

VARは多変量ARの自然な拡張で、係数が行列 $A_i$ になり、イノベーションの共分散行列 $\Sigma$ が同時刻ビン間の依存性を表す。

### 両方向推定によるギャップ補間（インペインティング）

```
|--- A ---||--- gap ---||--- B ---|
→→→→→→→→→                ←←←←←←←←←
  前向きAR予測              後ろ向きAR予測（時間反転）
           ↓           ↓
           両側からブレンドして埋める
```

- 前向きARと後ろ向きARの出力を距離重みでブレンド
- 状態空間のKalmanスムーザーがこの操作の理論的裏付けになる（前向きフィルタ＋後ろ向き平滑化）
- ノートを長くしたい場合はARで延長、短くしたい場合は単純にトリム → 対称な設計

## モデルの比較軸

| 手法 | 特徴 | 限界 |
|------|------|------|
| ビンごと独立AR | 実装簡単、Yule-Walker方程式で推定 | ビン間相関を無視 |
| VAR | ビン間の線形依存を捉える | K=80でパラメータ数爆発 |
| 正則化VAR | LassoでスパースなA_iを推定 | 正則化パラメータの選択が必要 |
| 動的因子モデル（状態空間） | 次元削減＋潜在構造の抽出 | M（潜在次元数）の設定が必要 |
| 位相ボコーダ（比較対象） | 実用的な標準手法 | メタリックなアーティファクトが出やすい |

## 評価方法

### 定量評価

- log-spectral distance（生成フレームと元フレームの比較）
- セグメンタルSNR
- spectral convergence

### 定性評価

- Griffin-Limで波形に戻して実際に聴く
- アーティファクトの種類を記述（ARモデル由来 vs 位相ボコーダ由来）

---

## 重要な設計上の注意点

- **局所定常性**: ARモデルは短時間（20〜30ms程度）の局所的な定常性を仮定。sustainのみに適用することでこの仮定を満たす
- **log-mel変換**: 振幅スペクトログラムはガウス分布の仮定に合わないため、log-melスペクトログラムを使用
- **再現可能性のための固定値**: n_fft、hop_length、メルビン数、AR次数の探索範囲、乱数シードをノートブック冒頭で明記
- **Yule-Walker方程式**: statsmodelsを使う場合も、方程式の導出と係数の意味を自分で説明できる状態にしておく（評価基準「プログラムを理解しているか」対策）
